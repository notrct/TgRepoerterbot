# 🚀 د ټیلیګرام راپور ورکوونکی بوټ

## 📋 شرح
دا پروګرام تاسې ته اجازه درکوي چې په ټیلیګرام کې سپمران، جعلي اکاونټونه او ناوړه چینلونه راپور کړئ.

## ✨ ځانګړتیاوې
- د کارن (User) او بوټ (Bot) دواړو ملاتړ
- د 9 څخه تر 999 پورې راپور لیږل
- د پرمختګ بار (Progress Bar)
- د وخت محاسبه
- ښایسته ډیزاین
- د انټرنیټ قطع کېدو سره مقاومت
